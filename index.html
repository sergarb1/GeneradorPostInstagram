GeneradorPostInstagram

Crea imágenes promocionales para Instagram desde tu terminal.
Funciona solo o con apoyo de opencode, Gemini CLI, Codex o Claude.

🐍 Python + Pillow 🤖 IA opcional 📸 1080×1080

⚡ Cómo funciona

📝

1. Configuras

Editas config.yaml con tu título, colores, features y mensaje. O dejas los valores genéricos de ejemplo.

🤖

2. (Opcional) IA

Escribes tu contexto en custom/context.md y ejecutas generate.sh --ai opencode. La IA genera el contenido del post automáticamente.

🖼️

3. Generas

Ejecutas ./generate.sh y obtienes un PNG 1080×1080 listo para Instagram en output/post.png.

🤖 Soporte para IA

El generador funciona sin IA con solo editar config.yaml. Pero si tienes una herramienta de IA instalada, puedes usarla para generar el contenido del post automáticamente a partir de una descripción de tu proyecto.

  • ▸
    opencode — generate.sh --ai opencode
  • ▸
    Gemini CLI — generate.sh --ai gemini
  • ▸
    OpenAI Codex CLI — generate.sh --ai codex
  • ▸
    Claude CLI — generate.sh --ai claude

Ejemplo context.md

# Mi Proyecto
App educativa gratuita para docentes.
Público: profesores de secundaria.
Tono: cercano e inspirador.
Valores: gratuito, colaborativo, código abierto.

La IA lee esto y genera el config.yaml automáticamente.

🚀 Inicio rápido

# Clonar
git clone https://github.com/sergarb1/GeneradorPostInstagram.git
cd GeneradorPostInstagram
# Generar (sin IA)
./generate.sh
# O con IA (necesitas opencode instalado)
echo "Mi proyecto mola" > custom/context.md
./generate.sh --ai opencode

📋 Requisitos

Programas necesarios

  • ▸
    Python 3.8+ — sudo apt install python3 python3-pip (Linux), brew install python (Mac), o python.org
  • ▸
    pip — Viene con Python 3.4+. Si falta: python3 -m ensurepip --upgrade
  • ▸
    Git — sudo apt install git (Linux), brew install git (Mac), o git-scm.com

Verifica: python3 --version && pip --version && git --version

Dependencias Python

# Con requirements.txt
pip install -r requirements.txt

Opcional — CLIs de IA

  • opencode — npm install -g @opencode/cli
  • Gemini CLI — pip install google-generativeai
  • Codex CLI — npm install -g @openai/codex
  • Claude CLI — npm install -g @anthropic-ai/claude

Las fuentes se descargan automáticamente. No requiere acción.
